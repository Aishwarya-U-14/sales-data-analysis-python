# sales-data-analysis-python
Week-2 Internship Project


This project analyzes sales data using Python and Pandas as part of Week 2 internship task.

## Project Overview
- Created a sales dataset with 100 records
- Included multiple products and categories
- Calculated total sales for each order
- Generated overall sales summary
- Analyzed product-wise, category-wise, and monthly sales performance
- Exported final output as a CSV file

## Key Insights
- Electronics and computer-related products contributed higher revenue
- High-value items such as laptops and smart watches generated significant sales
- Sales were recorded consistently across all months
- Product-wise and category-wise analysis helped identify top-performing segments

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab
